My first concern when i was getting this blog up, was to set up a code formatting plugin. I had another install of Â WordPress and there was no default plugin for that. So after typing some keywords in the plugin search came up with the Syntax Highlighter for WordPress which actually uses the SyntaxHighlighter . Searching around i found that Scott Hansleman uses the same plugin, so i decide to settled down with it.

code sample from Scott Hanselman's blog:

[csharp]
// Hello3.cs
using System;

public class Hello3
{
public static void Main(string[] args)
{
Console.WriteLine("Hello, World!");
Console.WriteLine("You entered the following {0} command line arguments:",
args.Length );
for (int i=0; i < args.Length; i++)
{
Console.WriteLine("{0}", args[i]);
}
}
}
